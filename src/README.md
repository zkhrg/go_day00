## Input format
only float or int number splitted by '\n'. if passed somthing else -> error and exit
when sequence is over insert '!end' and press enter.

#### Example 
```txt
1
2
3
4
5
!end
```
```txt
1.314
223.2
3123.000000
412
!end
```


## Format output
type intresting numbers of metrics w/o separator ONLY and press enter
#### Example 
```txt
1234
```
```txt
111111
```
```txt
1212
```
